# Fake News Detection using Machine Learning (Python)

This project detects whether a given news article is **REAL** or **FAKE**
using Machine Learning techniques in Python.

## Features
- Text preprocessing
- TF-IDF Vectorization
- Logistic Regression classifier
- Confidence-based prediction

## Files
- fake_news_detection.py – Main model and prediction logic
- merge_dataset.py – Dataset preprocessing script

## Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP

## Note
Datasets are not included in this repository.
